# Mural

Ao longo deste projeto resolvi implementar a função de excluir um post.
<h3> tecnologias usadas </h3>

<ul>
	<li> HTML </li>
	<li> Bootstrap </li>
	<li> Javascript </li>
	<li> Node JS </li>
</ul>

<h3> Funcionalidades </h3>

<ul>
	<li> Criar um novo post </li>
	<li> Exclui o post </li>
</ul>
